# 🎂 Happy Birthday — Rakshita (Kadimi)

A beautiful, animated birthday surprise webpage built with HTML, CSS, and JavaScript. 🎉

## 🌐 Live Demo

👉 [View the Birthday Page](https://vaibhav1817.github.io/Birthday-Wishes/)

---

## ✨ Features

- 🎊 Animated **"Happy Birthday"** text with letter-by-letter entrance
- 💌 Hidden **love letter** revealed on button click
- 🎈 Floating **balloon** animations
- 🌟 Twinkling **star decorations**
- 💫 Spinning **circular text** effect
- 🐱 Cute **mewmew GIF** inside the letter
- 💖 Heart animations throughout the page
- 📱 Responsive design

## 🚀 How to Run Locally

1. Clone the repository:
   ```bash
   git clone https://github.com/vaibhav1817/Birthday-Wishes.git
   ```

2. Open the project folder:
   ```bash
   cd Birthday-Wishes
   ```

3. Open `index.html` in your browser, or run a local server:
   ```bash
   python -m http.server 8080
   ```
   Then visit: `http://localhost:8080`
